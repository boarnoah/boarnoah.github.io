Generate coordinate and address data by placing markers on a map.
Originally created to assist with store location webscraping.

How to use:

Select desired format and data, then place markers on the map

doogal.co.uk has an assortment of geolocation related tools that will be handy with this data.
If this came in handy, please share it around. If it was broken I would appreciate if you could open a ticket here.

Not working/slow?

Address data requires Google Places API which has a maximum limit per ip and throttles for multiple request. If you desperately need to generate a large number of points, grab the source code for this page and change the API key's to your own. Contact me if you need help untangling the relevant sections form my Jekyll site.
