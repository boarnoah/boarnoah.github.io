Allows you to output coordinate + address data to JSON/CSV by placing markers on a map. Originally made to aid with some store location data scraping off websites.
Example uses:

  • You will probably know if you need it.
  • Creating a list of geolocated points to test/web-scrape an api that takes in location.
  • Ghetto target location for your doomsday device(s). Consider giving me a warning please.
doogal.co.uk has an assortment of geolocation related tools that will be handy with this data.
If this came in handy, please share it around. If it was broken I would appreciate if you could open a ticket here.

How to use:

Click to place markers, copy/download the coordinates at the bottom

Not working/slow?

Trying to include data that requires reverse geolocation (anything other than coordinates)? Google's service throttles the number of requests, and you may have run into this. If you desperately need to run this for a huge number of points, grab the source code and change the API key's to your own. Contact me if you need help untangling the relevant sections form my Jekyll site.
